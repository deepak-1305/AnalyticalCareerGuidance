Execution Steps Summary

Clone the Repository

Use GitHub URL:

git clone https://github.com/yourusername/career-guidance-system.git


Navigate into the project folder:

cd career-guidance-system


Install Node.js

Download and install LTS version from Node.js official site
.

Ensure node -v and npm -v return version numbers.

Install Backend Dependencies

Open terminal inside backend folder:

cd backend
npm install


Run Backend Server

Start server:

node server.js


Terminal shows: Backend running at http://localhost:3000.

Keep this terminal open.

Open Frontend

Open frontend/index.html in VS Code.

Right-click → Open with Live Server.

Start Assessment

Click “Start Assessment”.

Answer 20 questions sequentially.

View Results

After the last question, career scores appear as an animated bar chart.

Optional Updates

Pull updates from GitHub:

git pull


After editing files:

git add .
git commit -m "Update message"
git push


Notes

Backend is required for ML-based scoring.

For offline-only usage, scoring logic can run entirely in script.js.
